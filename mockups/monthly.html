Design notes

  • Categories are the primary axis, not weeks. The Excel grouped by week (I–V); this view groups by category. Weeks become a sub-grouping inside the drill-down if needed.
  • Variance number is right-aligned, color-coded. Single most-scannable signal: red = over, yellow = within 10% of plan, green = under. The progress bar reinforces with the same color.
  • Inline budget editing. Tap a planned number → it becomes a number input with a brand-color border + ring, the row gets a soft brand background to confirm "you're editing this." Save on blur or Enter.
  • Expansion is in-place, not a sheet. Tapping the chevron (or anywhere on the row) reveals transactions inline with a vertical guide line. No navigation; the user keeps their place in the list.
  • Tap-through to drill-down. Tapping anywhere in a category row (other than the planned-amount edit zone) navigates to the Category drill-down screen. The expanded preview is a teaser.
  • Group categories collapse via "5 sub" hint. Top-level groups (OSTALO, KARTICE, MI) show their aggregated total. Drilling in shows each sub-category individually. Reduces list length without hiding data.
