How to consume these tokens

  • Tailwind utility: every --color-* token automatically generates bg-*, text-*, border-*, ring-* utilities. Same for radius, shadow, font, text-size.
  • Raw CSS: reference as var(--color-bg) when you need them outside utility contexts (keyframes, gradients).
  • Figma import: paste the @theme block into the Tokens Studio plugin → import as variables. Light + dark sets become two modes on the same variable collection.
  • Angular: copy the @theme + :where(.dark) blocks into src/styles.css; every component template uses the same utility classes shown in these mockups.
  • Source of truth: tokens.css in this folder. The HTML files inline the same block for browser-CDN compatibility.
Tokens are stable contracts. Naming is semantic, not literal — --color-success-700 never becomes --color-green-700 even if the green changes.
