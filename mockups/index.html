How to view

  1. Open this folder via a local server: cd mockups && python3 -m http.server 8000 then visit http://localhost:8000.
  2. Or open index.html directly — most browsers will load it. If shared tokens fail to apply over file://, fall back to the local server.
  3. Toggle dark mode with the button in the top-right.
  4. Each screen is self-contained HTML — open directly to inspect, copy snippets, or paste into Figma via the html.to.design plugin.
Built with Tailwind v4 · Inter · Lucide · designed for Angular 20+ PWA + Firebase.
