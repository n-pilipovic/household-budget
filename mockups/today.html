Desktop replaces the FAB with a persistent Quick Add input in the top bar — same parser, same flow. The user attribution chip on the left of the input is the always-visible reminder of who's logging. Press N from anywhere to focus it.

Design notes

  • User attribution chip: always 8px-radius circle with the user's initial, in their color (Novica blue / Nada rose). Pure brand-coded — we never label "by Novica" in text.
  • Just-synced state: a row that just arrived from the partner gets a 2-second tinted background in the partner's user color, plus a 2px left bar. Faded out after the animation completes; here we're showing the mid-state.
  • Numbers: tabular-nums everywhere amounts appear. RSD is shown as a faint suffix, not a code prefix — keeps the number itself the visual anchor.
  • No bottom nav on mobile: the app has only 2 destinations (Today, Monthly). A footer link is enough — no need to spend persistent screen real estate.
  • Variance pill: green when on pace, yellow when within 10% of plan, red when over. Single semantic signal, never a multi-color chart.
